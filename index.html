🚀 什麼是 Claude Code？

Claude Code 是 Anthropic 推出的革命性命令行工具，它允許開發者直接在終端機中與 Claude 3.5 Sonnet 模型協作。這不是簡單的聊天機器人，它可以讀取你的檔案、執行終端指令、並協助你重構代碼。

直接操作終端

無需切換視窗，Claude 直接在你的 CLI 環境中工作。

理解專案結構

它可以分析整個文件夾，理解代碼上下文。

📥 下載與系統需求

前置需求

  • Node.js 18.0.0 或更高版本
  • 有效的 Anthropic API Key (需綁定信用卡)
  • Git (版本控制)

macOS 用戶 (推薦)

使用 Homebrew 安裝是最快的方式：

brew install anthropic/tap/claude-code

Windows / Linux (透過 npm)

確保您已經安裝了 Node.js，然後執行：

npm install -g @anthropic-ai/claude-code

⚙️ 初始化與授權

安裝完成後，您需要進行首次登入設定。請在終端機輸入以下指令：

1. 啟動 Claude Code

claude

2. 完成授權

系統會自動開啟瀏覽器引導您登入 Anthropic 帳號並授權。授權完成後，您將獲得一組 Token，終端機會自動偵測並完成設定。

❓ 常見問題

Q: Claude Code 是免費的嗎？

工具本身是開源或免費下載的，但使用時會消耗 Anthropic API 的 Token，這部分是需要付費的。

Q: 支援中文對話嗎？

支援！你可以直接用中文下指令，例如「幫我解釋這個檔案在做什麼」。

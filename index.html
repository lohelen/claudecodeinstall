🚀 什麼是 Claude Code？

Claude Code 是 Anthropic 推出的革命性命令行工具，直接在終端機中與最新的 Claude 4.6 模型協作。它可以讀取檔案、執行指令、跑測試並協助重構代碼。

❓ 常見問題

Q: Claude Code 是免費的嗎？

工具本身免費，但會消耗 Anthropic API Token。Pro/Max 方案提供額外的存取權限與更高等級的模型。

Q: 操作介面是在哪裡？

它是一款「終端機原生」工具，在命令列運行。也可以在 VS Code 的內建終端機中使用。
